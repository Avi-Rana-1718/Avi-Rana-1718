C

By Avi Rana

C is a programming language by Dennis Ritchie in 1972. It became popular because it was considered easy & simple. C++, C# & Java are superset of C. It is a major part of operating systems like Windows, UNIX, Linux & Android. Also device drivers, video games and microprocessors are written exclusively in C. These program are required to be fast and occupy the least amount of memory, hence they are also written in C. The standard version of C is C99.

A C program consists of algorithms and statements.

Programs

A program is a collection of algorithms or implementation of one or more algorithms. A program is written in a computer language(or low level language) which the machine turns into Machine Code also known as Binary Code or Object Code (index.obj).

Algorithms

An algorithms is a combination of instructions that must be excecuted in a particular sequence. It can be expressed in a normal human understandable way. For example, baking a cake, finding compound interest or an algorithm for turning on a fan switch
  • Step 1 - Get Up
  • Step 2 - Walk upto the switch board
  • Step 3 - Determine which switch turns the fan on
  • Step 4 - Press the switch
In short, an algorithm can de defined as a systematic & logical approach for solving a problem.

Statements

A C program consists of a number of steps, these steps are known as Statements. They are fragments of the C program that are executed in sequence. It is a followed by a semicolon at the end.

Getting Started With C

To make a C program, we must first write it's source code.

Source Code is a fundamental part of a program. It is usually created by a programmer. It can be understood by humans. This source code is converted into object code which essentailly makes up the program. It undergoes the following process to finally became the object code:
  • Source code
  • Pre processor
  • Expanded Code - Include Header Files
  • Compiler
  • Assembly Code
  • Assembler
  • Machine Code/Object Code
  • Linker/Loader
  • test.exe
